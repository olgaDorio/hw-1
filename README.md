# Как запустить

```sh
npm install
```

## Сборка фронта (в директорию `/docs`)
```sh
npm run dev
npm run build
```

## Запуск сервера на порту, указанном в `process.env.PORT` (или 8000)
```sh
npm run server
```

## Запуск линтеров

```sh
npm run lint
```
